# Quick Start Checklist

Use this checklist to set up your WalkScape Home Assistant integration.

## Before You Begin

- [ ] Home Assistant is installed and running
- [ ] HACS is installed in Home Assistant
- [ ] You have a WalkScape account
- [ ] Your WalkScape profile sharing is enabled

## Installation Steps

- [ ] Copy your WalkScape character ID from your shared profile URL
- [ ] Open HACS in Home Assistant
- [ ] Add this repository as a custom repository in HACS
- [ ] Install the WalkScape integration through HACS
- [ ] Restart Home Assistant
- [ ] Go to Settings ‚Üí Devices & Services
- [ ] Click "+ Add Integration"
- [ ] Search for "WalkScape"
- [ ] Enter your character ID
- [ ] Verify all sensors are created

## Post-Installation

- [ ] Check that sensors are showing data
- [ ] Add sensors to your dashboard
- [ ] (Optional) Set up automations
- [ ] (Optional) Create history graphs
- [ ] (Optional) Add custom Lovelace cards

## Troubleshooting

If something doesn't work:

1. Check Home Assistant logs (Settings ‚Üí System ‚Üí Logs)
2. Verify your character ID is correct
3. Ensure profile sharing is enabled in WalkScape
4. Try restarting Home Assistant again
5. Check GitHub issues for known problems
